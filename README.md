# nim-preloader

## Install

```bash
npm install --save nim-preloader
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'nim-preloader'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
